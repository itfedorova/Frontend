Нежданчик!!! Куда же делся отступ сверху и снизу зеленого блока?
Происходит схлопывание. Способы исправить:
  1. Добавить border к внешнему DIV (рамка должа быть быть видимой)
  2. Добавить padding к внешнему DIV (минимум 1px)
  3. Использовать position:absolute у внешнего DIV
  4. Использовать свойство float: left иди right
  5. Использовать строчные элемент в качестве внешнего (свойство display: inline)
    6. Не схлопываются отступы, указанные для тега html.
